# US-18: No error messages are shown when submitting the "Alquiler" form with required fields empty or invalid

# Key details

## Severity
🟠 Major

## Priority
🟧 High

## Environment
Opera 132, 1280x720 (Chrome blocked by [US-1](./US-1.md))

## Component
Make Order - "Alquiler" Form

## Description

### Preconditions
1. Go to the home page in Opera.
2. Click "Pedir".
3. Fill in all fields with valid data and click "Siguiente".

### Steps to reproduce
1. Leave "Periodo de alquiler" empty and select an invalid delivery date.
2. Click "Pedir".

### Expected result
- Invalid fields are visually highlighted.
- An error message appears for each invalid field.

### Actual result
- There is no visual error indication on the fields.
- No error message appears.
- No other action occurs (the screen remains static).
