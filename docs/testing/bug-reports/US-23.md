# US-23: Campo “Dirección” con 50 caracteres se marca como inválido y no permite avanzar al siguiente formulario

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟨 Medium

## Entorno
- Chrome 149, 1920x1080

## Componente
Realizar Pedido - Formulario "Para quién es el scooter"

## Descripción

### Precondiciones
1. Ingresar a la página de inicio en Chrome 149.
2. Hacer clic en “Pedir”.
3. Ingresar datos válidos en todos los campos excepto “Apellido“.

### Pasos para reproducir
1. Ingresar `350 East 1st Street Apt. 205, Los Angeles, CA 9001`.
2. Hacer clic en “Siguiente”.

### Resultado esperado
- El campo no se resalta como inválido.
- No aparece mensaje de error.
- El sistema puede avanza al siguiente formulario.

### Resultado actual
- El campo muestra se resalta como invalido.
- Aparece mensaje de error.
- No se muestra el siguiente formulario al avanzar.
