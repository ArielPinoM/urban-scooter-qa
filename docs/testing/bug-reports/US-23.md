# US-23: "Dirección" field with 50 characters is marked invalid and does not allow advancing to the next form

# Key details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Go to the home page in Chrome 149.
2. Click "Pedir".
3. Enter valid data in all fields except "Apellido".

### Steps to reproduce
1. Enter `350 East 1st Street Apt. 205, Los Angeles, CA 9001`.
2. Click "Siguiente".

### Expected result
- The field is not highlighted as invalid.
- No error message appears.
- The system can advance to the next form.

### Actual result
- The field is marked invalid.
- An error message appears.
- The next form does not appear when advancing.
