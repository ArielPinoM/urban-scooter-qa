# US-29: Campo "Comentario" permite caracteres especiales

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟩 Low

## Entorno
- Chrome 149, 1920x1080

## Componente
Realizar Pedido - Formulario "Para quién es el scooter"

## Descripción

### Precondiciones
1. Ingresar a la página de inicio en Chrome 149.
2. Hacer clic en “Pedir”.
3. Ingresar datos válidos en todos los campos y hacer clic en “Siguiente”.
4. Seleccionar datos válidos en los campos de “Fecha de entrega” y “Periodo de alquiler”.

### Pasos para reproducir
1. Escribir `№%@\"` en el campo “Comentario”.
2. Hacer clic en “Pedir”.

### Resultado esperado
- El campo se resalta como inválido.
- Aparece mensaje de error.
- No es posible completar el pedido.

### Resultado actual
- El campo no muestra indicio de error.
- No se muestra mensaje de error.
- Es posible completar el pedido.
