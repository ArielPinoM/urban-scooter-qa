# US-27: "Comentario" field allows more than 26 characters

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Navigate to the home page in Chrome 149.
2. Click the "Pedir" button.
3. Enter valid data in all fields and click "Siguiente".
4. Select valid data in the "Fecha de entrega" and "Periodo de alquiler" fields.

### Steps to reproduce
1. Type `ArielArielArielArielArielA` (26 characters) in the "Comentario" field.
2. Click the "Pedir" button.

### Expected result
- The field is highlighted to indicate an invalid state.
- An error message is displayed.
- Order submission is prevented.

### Actual result
- The field displays no error indication.
- No error message is displayed.
- Order submission is allowed.
