# US-38: POST api/v1/courier devuelve 201 Created al mandar campo login con caracteres numéricos

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟨 Medium

## Entorno
- API Ez-scooter v1.0.0
- Postman 12.17.3

## Componente
API - Crear un mensajero

## Descripción

### Precondiciones

- El servidor de back-end está en ejecución.

- No existe previamente un mensajero con el login `1234567890`.

### Pasos para reproducir

1. Crear una solicitud POST a {{base_url}}/api/v1/courier.

2. Configurar el cuerpo con los siguientes datos:

```json
{
  "login": "1234567890",
  "password": "1234"
}
```

3. Enviar la solicitud.

### Resultado esperado
- Código de estado 400 Bad Request.

### Resultado actual
- Código de estado 201 Created.
- Cuerpo de respuesta: {“ok“: true}
