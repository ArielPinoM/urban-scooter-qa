# US-32: "Clave o usuario inválido" popup displayed when tapping "Iniciar sesión" without internet connection

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Android Emulator: Galaxy A3, Android 9

## Component
Mobile - No Internet Access - Popup

## Description

### Preconditions

Airplane mode is enabled on the device/emulator.

### Steps to reproduce
1. Open the application.
2. Tap the "Iniciar sesión" button.

### Expected result
A "Sin acceso a Internet" popup is displayed.

### Actual result
Screenshot showing the "Clave o usuario inválido" popup displayed with airplane mode enabled.
