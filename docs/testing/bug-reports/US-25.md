# US-25: "Teléfono" field without the initial "+" sign is not marked invalid and allows advancing

# Key details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Go to the home page in Chrome 149.
2. Click "Pedir".
3. Enter valid data in all fields except "Teléfono".

### Steps to reproduce
1. In "Teléfono", enter 12345678901.
2. Click "Siguiente".

### Expected result
- The field is highlighted as invalid.
- An error message appears.
- The form does not advance when clicking "Siguiente".

### Actual result
- The field shows no error indicator.
- The form advances to "Alquiler" without any warning.
