# US-15: If "Teléfono" is empty or invalid, trying to advance shows error "Introduce un número de teléfono válido".

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Opera 132, 1920x1080

## Component
Home Page - Header

## Description

### Preconditions
1. Start the web application in Opera.
2. Click "Pedir".

### Steps to reproduce
1. Click "Siguiente".
2. Observe the error message in the "Teléfono" field.

### Expected result
It shows the error message "Introduce un número de teléfono válido".

### Actual result
It shows the error message "Introduce un número válido".
