# US-65: POST api/v1/courier devuelve 201 Created al mandar campo firstName con caracteres latinos con puntos

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟨 Medium

## Entorno
- API Ez-scooter v1.0.0
- Postman 12.17.3

## Componente
API - Crear un mensajero

## Descripción

### Precondiciones

- El servidor de back-end está en ejecución.

- No existe previamente un mensajero con el login `Ariel`.

### Pasos para reproducir

1. Crear una solicitud POST a {{base_url}}/api/v1/courier.

2. Configurar el cuerpo con los siguientes datos:

```json
{
  "login": "Ariel",
  "password": "1234",
  "firstName": "ar.iel"
}
```

3. Enviar la solicitud.

### Resultado esperado
- Código de estado 400 Bad Request.

### Resultado actual
- Código de estado 201 Created.
- Cuerpo de respuesta: {“ok“: true}
