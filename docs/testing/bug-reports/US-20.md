# US-20: "Apellido" field with 16 characters is not marked invalid and allows advancing in the form

# Key details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Go to the home page in Chrome 149.
2. Click "Pedir".
3. Enter valid data in all fields except "Apellido".

### Steps to reproduce
1. Enter `PinoPinoPinoPino`.
2. Click "Siguiente".

### Expected result
- The field is highlighted as invalid.
- An error message appears.
- The system does not advance to the next form.

### Actual result
- The field shows no error indicator.
- The next form is displayed.
