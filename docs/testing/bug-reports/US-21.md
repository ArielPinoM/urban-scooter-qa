# US-21: Campo “Apellido” con 17 caracteres no se marca como inválido y permite avanzar en el formulario

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟨 Medium

## Entorno
- Chrome 149, 1920x1080

## Componente
Realizar Pedido - Formulario "Para quién es el scooter"

## Descripción

### Precondiciones
1. Ingresar a la página de inicio en Chrome 149.
2. Hacer clic en “Pedir”.
3. Ingresar datos válidos en todos los campos excepto “Apellido“.

### Pasos para reproducir
1. Ingresar `PinoPinoPinoPinoP`.
2. Hacer clic en “Siguiente”.

### Resultado esperado
- El campo se resalta como inválido.
- Aparece mensaje de error.
- El sistema no avanza al siguiente formulario.

### Resultado actual
- El campo no muestra ningún indicador de error.
- Se muestra el siguiente formulario.
