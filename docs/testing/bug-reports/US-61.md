# US-61: POST api/v1/courier returns 201 Created when sending firstName field with 13 Latin characters

# Key Details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- Ez-scooter API v1.0.0
- Postman 12.17.3

## Component
API - Create Courier

## Description

### Preconditions

- Back-end server is running.

- A courier with login `Ariel` does not already exist.

### Steps to Reproduce

1. Create a POST request to {{base_url}}/api/v1/courier.

2. Configure the request body with the following data:

```json
{
  "login": "Ariel",
  "password": "1234",
  "firstName": "ArielArielAri"
}
```

3. Send the request.

### Expected Result
- Status code 400 Bad Request.

### Actual Result
- Status code 201 Created.
- Response body: {"ok": true}
