# US-24: Empty "Dirección" field does not show an error and allows advancing to the next form

# Key details

## Severity
🔵 Minor

## Priority
🟧 High

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Go to the home page in Chrome 149.
2. Click "Pedir".
3. Enter valid data in all fields except "Dirección".

### Steps to reproduce
1. Select the address field.
2. Remove focus from the field.
3. Click "Siguiente".

### Expected result
- The field is highlighted as invalid.
- An error message appears.
- The system does not allow advancing to the "Alquiler" form.

### Actual result
- The field shows no error indication.
- The system allows advancing to the "Alquiler" form without any warning.
