# US-24: Campo "Dirección" vacío no muestra error y permite avanzar al siguiente formulario

# Detalles clave

## Severidad
🔵 Minor

## Prioridad
🟧 High

## Entorno
- Chrome 149, 1920x1080

## Componente
Realizar Pedido - Formulario "Para quién es el scooter"

## Descripción

### Precondiciones
1. Ingresar a la página de inicio en Chrome 149.
2. Hacer clic en “Pedir”.
3. Ingresar datos válidos en todos los campos excepto “Dirección“.

### Pasos para reproducir
1. Seleccionar el campo dirección.
2. Quitar el foco del campo.
3. Hacer clic en “Siguiente”.

### Resultado esperado
- El campo se resalta como inválido.
- Aparece mensaje de error.
- El sistema no permite avanzar al formulario “Alquiler”.

### Resultado actual
- El campo no muestra ninguna señal de error.
- El sistema permite avanzar al formulario “Alquiler” sin ningún aviso.
