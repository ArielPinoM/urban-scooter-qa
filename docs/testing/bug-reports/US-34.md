# US-34: POST api/v1/courier returns 201 Created when sending login field with a single character

# Key details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- API Ez-scooter v1.0.0
- Postman 12.17.3

## Component
API - Create Courier

## Description

### Preconditions

- The back-end server is running.

- No courier with login `A` exists previously.

### Steps to reproduce

1. Create a POST request to {{base_url}}/api/v1/courier.

2. Set the request body with the following data:

```json
{
  "login": "A",
  "password": "1234"
}
```

3. Send the request.

### Expected result
- Status code 400 Bad Request.

### Actual result
- Status code 201 Created.
- Response body: {"ok": true}
