# US-11: The "Estado del pedido" button changes color on hover, unlike what is specified in Figma

# Key details

## Severity
⚪ Trivial

## Priority
🟩 Low

## Environment
- Chrome 149, 1920x1080
- Chrome 149, 1280x720
- Opera 132, 1920x1080
- Opera 132, 1280x720

## Component
Home Page - Header

## Description
The Figma design does not show a color change in the text when hovering.

### Preconditions
Open the application in any environment.

### Steps to reproduce
1. Hover over the "Estado del pedido" button.
2. Observe the button text color.

### Expected result
The button text color does not change.

### Actual result
The button text color changes.
