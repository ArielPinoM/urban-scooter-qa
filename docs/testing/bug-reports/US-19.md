# US-19: The "Fecha de entrega" placeholder does not match the design.

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Chrome 149, 1920x1080
- Opera 132, 1920x1080

## Component
Make Order - "Alquiler" Form

## Description

### Preconditions
Go to the home page in both environments.

### Steps to reproduce
1. Click "Pedir".
2. Fill in the form fields with valid data and click "Siguiente".
3. Observe the placeholder text in the "Fecha de entrega" field.

### Expected result
The placeholder text matches "* Cuándo entregar el scooter".

### Actual result
The placeholder text says "* Fecha de entrega".
