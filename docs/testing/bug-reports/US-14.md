# US-14: If "Estación de metro" is empty, trying to advance shows error "Introduce un Estación de metro correcto".

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Opera 132, 1920x1080

## Component
Home Page - Header

## Description

### Preconditions
1. Start the web application in Opera.
2. Click "Pedir".

### Steps to reproduce
1. Click "Siguiente".
2. Observe the error message in the "Estación de metro" field.

### Expected result
It shows the error message "Introduce una estación de metro correcta".

### Actual result
It shows the error message "Elige la estación".
