# US-17: Campo "Periodo de alquiler" vacío no se marca como inválido y no muestra error al hacer clic en "Pedir"

# Detalles clave

## Severidad
🟠 Major

## Prioridad
🟧 High

## Entorno
- Opera 132, 1920x1080

## Componente
Realizar Pedido - Formulario "Alquiler"

## Descripción

### Precondiciones
1. Ingresar a la página de inicio con Opera.
2. Llenar el formulario “Para quién es el scooter“ con datos válidos y hacer clic en “Siguiente”.

### Pasos para reproducir
1. Seleccionar una fecha de entrega válida.
2. Dejar el campo “Periodo de alquiler” vacío.
3. Hacer clic en “Pedir”.

### Resultado esperado
- El campo se resalta como inválido.
- Aparece mensaje de error “Introduce un periodo de alquiler correcto”.
- No se abre ventana de confirmación.

### Resultado actual
- El campo no muestra indicador visual de error.
- No aparece mensaje de error.
- La pantalla permanece igual.
