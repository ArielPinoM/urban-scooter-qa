# US-17: Empty "Periodo de alquiler" field is not marked invalid and does not show an error when clicking "Pedir"

# Key details

## Severity
🟠 Major

## Priority
🟧 High

## Environment
- Opera 132, 1920x1080

## Component
Make Order - "Alquiler" Form

## Description

### Preconditions
1. Open the home page in Opera.
2. Fill out the "Para quién es el scooter" form with valid data and click "Siguiente".

### Steps to reproduce
1. Select a valid delivery date.
2. Leave the "Periodo de alquiler" field empty.
3. Click "Pedir".

### Expected result
- The field is highlighted as invalid.
- The error message "Introduce un periodo de alquiler correcto" appears.
- No confirmation popup opens.

### Actual result
- The field does not show a visual error indicator.
- No error message appears.
- The screen remains unchanged.
