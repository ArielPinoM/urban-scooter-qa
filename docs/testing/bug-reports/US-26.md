# US-26: "Comentario" field allows 25 characters

# Key details

## Severity
🔵 Minor

## Priority
🟩 Low

## Environment
- Chrome 149, 1920x1080

## Component
Make Order - "Para quién es el scooter" Form

## Description

### Preconditions
1. Go to the home page in Chrome 149.
2. Click "Pedir".
3. Enter valid data in all fields and click "Siguiente".
4. Select valid data in the "Fecha de entrega" and "Periodo de alquiler" fields.

### Steps to reproduce
1. Type `ArielArielArielArielAriel` in the "Comentario" field.
2. Click "Pedir".

### Expected result
- The field is highlighted as invalid.
- An error message appears.
- It is not possible to complete the order.

### Actual result
- The field shows no error indication.
- No error message appears.
- It is possible to complete the order.
