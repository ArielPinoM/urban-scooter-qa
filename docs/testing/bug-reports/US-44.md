# US-44: POST api/v1/courier returns 201 Created when sending login field with characters from other languages

# Key details

## Severity
🔵 Minor

## Priority
🟨 Medium

## Environment
- API Ez-scooter v1.0.0
- Postman 12.17.3

## Component
API - Create Courier

## Description

### Preconditions

- Back-end server is running.

- No courier with login `加布里埃爾F加布爾` exists previously.

### Steps to reproduce

1. Create POST request to {{base_url}}/api/v1/courier.

2. Set request body with data:

```json
{
  "login": "加布里埃爾F加布爾",
  "password": "1234"
}
```

3. Send request.

### Expected result
- Status code 400 Bad Request.

### Actual result
- Status code 201 Created.
- Response body: {"ok": true}
