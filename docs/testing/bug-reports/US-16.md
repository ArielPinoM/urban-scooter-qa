# US-16: Missing delivery date validation: orders are allowed with invalid dates

# Key details

## Severity
🟠 Major

## Priority
🟧 High

## Environment
- Opera 132, 1920x1080

## Component
Make Order - "Alquiler" Form

## Description

### Preconditions
1. Open the home page in Opera.
2. Click "Pedir".
3. Fill out the "Para quién es el scooter" form with valid data and click "Siguiente".

### Steps to reproduce
1. Select today's date in the "Fecha de entrega" field.
2. Select "un día" in the "Periodo del alquiler" field.
3. Click "Pedir".

### Expected result
An error message should appear: "Introduce una fecha de entrega correcta". The confirmation popup should not appear.

### Actual result
No error message appears. The popup "¿Desea hacer un pedido?" appears.
